# Template Preview Images

This directory contains preview images for the 10 OPAC templates.

## Required Images (1200x800px recommended)

1. `modern-minimal.jpg` - Modern Minimal template
2. `classic-library.jpg` - Classic Library template
3. `vibrant-colorful.jpg` - Vibrant & Colorful template
4. `dark-elegant.jpg` - Dark Elegant template
5. `academic-professional.jpg` - Academic Professional template
6. `kids-friendly.jpg` - Kids Friendly template
7. `magazine-style.jpg` - Magazine Style template
8. `minimalist-zen.jpg` - Minimalist Zen template
9. `bold-geometric.jpg` - Bold Geometric template
10. `nature-inspired.jpg` - Nature Inspired template

## How to Create Preview Images

1. Take screenshots of each template in action
2. Crop to 1200x800px (3:2 aspect ratio)
3. Optimize for web (80-90% JPEG quality)
4. Name files exactly as listed above
5. Place in this directory

## Temporary Placeholders

Until actual screenshots are available, you can use placeholder images or design mockups that represent each template's visual style.
